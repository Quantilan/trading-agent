# Privacy Policy

**Last updated:** March 24, 2026

## 1. Introduction

Quantilan ("we," "us," "our") respects your privacy. This Privacy Policy explains how we collect, use, and protect your personal information when you use our trading bot service.

## 2. Information We Collect

### 2.1 Information You Provide

**Telegram Account:**
- Telegram user ID
- Telegram username
- Chat ID for bot communication
- Language preference

**Subscription Data:**
- Selected plan (Start/Pro)
- Payment transaction IDs
- License keys generated
- Subscription start/end dates

**Support Communications:**
- Messages sent to @quantilan_support
- Technical issue reports
- Feature requests

### 2.2 Automatically Collected

**Usage Data:**
- Agent online/offline status
- Last connection timestamp
- Command usage statistics (e.g., /status, /positions)
- Error logs (anonymized)

**Technical Information:**
- IP address (for API requests)
- Device type (VPS, Mac Mini, etc.)
- Software version
- Exchange connected (Binance, Bybit, etc.)

### 2.3 What We DO NOT Collect

**We NEVER collect:**
- ❌ Your exchange API keys
- ❌ Your exchange secret keys
- ❌ Your private keys
- ❌ Trading account balances
- ❌ Personal identification documents
- ❌ Bank account information

## 3. How We Use Your Information

We use collected data for:

**Service Delivery:**
- Providing access to the trading agent
- Managing your subscription
- Generating license keys
- Monitoring agent status

**Communication:**
- Sending trading notifications via Telegram
- Responding to support requests
- Broadcasting important updates
- Service maintenance notifications

**Improvement:**
- Analyzing usage patterns
- Fixing bugs and errors
- Developing new features
- Performance optimization

**Security:**
- Detecting fraudulent activity
- Preventing abuse
- Protecting user accounts
- Compliance with laws

## 4. Legal Basis for Processing

If you are in the EU/EEA, we process your data based on:
- **Contract performance**: To provide the service you requested
- **Legitimate interests**: To improve and secure our service
- **Consent**: Where you have explicitly consented

## 6. Data Storage and Security

### 6.1 Storage

- Data stored on secure servers
- Encrypted databases
- Regular security audits
- Access limited to authorized personnel

### 6.2 Retention

We retain your data:
- **Active subscriptions**: Duration of subscription + 12 months
- **Expired subscriptions**: 24 months for legal compliance
- **Usage logs**: 90 days
- **Support messages**: 24 months

### 6.3 Security Measures

- SSL/TLS encryption for data transmission
- Database encryption at rest
- Regular security updates
- Access logging and monitoring
- Two-factor authentication for admin access
